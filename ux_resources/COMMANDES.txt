1. MODE REPRISE (ou MODE AUTONOME)
   Fichier: prompts/PROMPT_REPRISE.md

   QUAND: Tu reprends le projet apres une pause
          OU t'es fatigue et tu veux juste valider

   COMMENT:
   - Dis "Mode reprise" ou "Mode autonome"
   - Claude relit le contexte et les regles UX
   - Si fatigue: il propose, tu dis "ok" ou "non"
   - Pret a bosser

   RACCOURCIS (mode autonome):
   | Tu dis    | Claude fait                    |
   |-----------|--------------------------------|
   | "ok"/"go" | Execute les propositions       |
   | "non"     | Arrete et demande quoi changer |
   | "push"    | Commit + push GitHub           |
   | "relance" | Restart serveur                |

--------------------------------------------

2. MODE CSS FIX
   Fichier: prompts/PROMPT_CSSFIX.md

   QUAND: Bug visuel (alignement, spacing, overflow)

   COMMENT:

   - Claude liste les problemes
   - Tu dis "go"
   - Il corrige tout d'un coup

4. MODE INTEGRATION DEEP RESEARCH
   Fichier: prompts/PROMPT_INTEGRATION_DEEP_RESEARCH.md

   QUAND: Tu recois les resultats de ChatGPT Deep Research

   COMMENT:
   - Dis "integre la deep research" + colle les resultats
   - Claude detecte les doublons avec WEB.md/MOBILE.md
   - Claude complete avec WebSearch si besoin
   - Claude integre proprement section par section
   - Met a jour les compteurs

   RACCOURCIS:
   | Tu dis              | Claude fait                        |
   |---------------------|-----------------------------------|
   | "integre tout"      | Workflow complet                   |
   | "verifie doublons"  | Juste detection doublons           |
   | "complete [section]"| WebSearch + integration 1 section  |

   ANTI-DOUBLONS: Claude grep avant d'ajouter quoi que ce soit
